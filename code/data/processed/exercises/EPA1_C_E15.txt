En español, el gerundio (verbos terminados en "-ando" o "-endo") se usa para expresar una acción que ocurre al mismo tiempo que otra o para explicar cómo se realiza una acción. En cambio, los adverbios de relativo (p. ej., “cuando”, “quien”, “donde”) se utilizan para introducir información adicional sobre un lugar, tiempo, persona o circunstancia.
   * Ejemplos:

      * “Íbamos a cenar juntas una vez a la semana, donde nos turnamos para elegir el restaurante.”
(La información adicional proporcionada no es sobre un lugar.)
      * "Íbamos a cenar juntas una vez a la semana, turnándonos para elegir el restaurante."
(Describe cómo se turnan para cumplir con esta costumbre de cenar juntas.)
      * “Visitamos un pequeño pueblo en la montaña, donde cada calle parecía sacada de una postal antigua.”
(Se proporciona información adicional sobre el pueblo.)
Advertencia: el gerundio no se puede utilizar para expresar acciones que ocurren después de la acción principal ni para expresar características permanentes o impresiones generales de algo ya mencionado.
         * Ejemplos:

            * "Salió de casa, llegando a la oficina a las 9:30 h."
(Llegar es una acción posterior a salir de casa.)
            * “Visitamos un pequeño pueblo en la montaña, pareciendo cada calle sacada de una postal antigua.”
(Se explica una característica permanente del pueblo.)
