Práctica
Completa las siguientes oraciones eligiendo la opción correcta entre reclamar o reembolsar. Luego, explica por qué has elegido esa opción.


   1. Después de recibir el paquete equivocado, decidí __________ mi dinero.
   * a) reclamar
   * b) reembolsar

      2. El cliente fue a la tienda para __________ el precio del producto que había comprado porque estaba defectuoso.
      * a) que le reclamaran
      * b) que le reembolsaran

         3. Cuando cancelaron el evento, pedí que me __________ el importe de las entradas.
         * a) reclamaran
         * b) reembolsaran

            4. Si tienes algún problema con el servicio, puedes __________ en atención al cliente.
            * a) reclamar
            * b) reembolsar

               5. Al no recibir el pago, decidieron __________ el dinero que se les debía.
               * a) reclamar
               * b) reembolsar
