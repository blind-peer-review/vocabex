Práctica
Relaciona las siguientes descripciones con las palabras “poste” y “publicación”.
* Algo que se comparte en redes sociales, como Facebook o Instagram, que puede incluir texto, fotos o vídeos.
* Estructura de madera o metal que se utiliza para sostener algo, como cables de electricidad, señales de tráfico o una lámpara en la calle.
* Un artículo, foto o video que una persona sube a su blog o página web para que otros lo vean.
* Pieza vertical utilizada en la construcción para soportar una estructura o para delimitar un área.
* El acto de compartir una opinión, comentario o noticia en un foro de internet.
