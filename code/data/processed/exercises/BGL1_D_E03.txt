Práctica
Este ejercicio se centra en la práctica de traducciones literales inglés-español que son inadecuadas debido a las diferencias estructurales entre ambos idiomas. Lee las frases que se presentan a continuación. Tu tarea es corregir la traducción incorrecta al español para que resulte más idiomática.
Ejemplo
Inglés: “I don’t think it’s very interesting or useful to evaluate everything, like all these little details”.
Traducción incorrecta: “No creo que sea muy interesante ni útil evaluarlo todo, como todos estos pequeños detalles”.
Corrección: “No creo que sea muy interesante ni útil evaluarlo todo, es decir, cada pequeño detalle”.
