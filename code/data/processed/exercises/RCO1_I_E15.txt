Práctica
Lee cada par de oraciones. Decide si es más adecuado usar “que” o “lo que” para conectar las oraciones siguientes, basándote en la información proporcionada en la primera. A veces, podrías necesitar reorganizar ligeramente las últimas oraciones para que el uso de “que” o “lo que” sea gramaticalmente correcto.
Ejemplo
* Oración 1: Este restaurante está ubicado cerca del parque central.
* Oración 2: Esto es ideal para disfrutar de una comida tranquila después de un paseo por el parque.
* Oración completa: Este restaurante está ubicado cerca del parque central, lo que lo hace ideal para disfrutar de una comida tranquila después de un paseo por el parque.
