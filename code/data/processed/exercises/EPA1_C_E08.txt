Práctica
Completa las siguientes frases con combinaciones de expresiones temporales usando la estructura correcta. Fíjate bien en el orden de las palabras.
Ejemplos
(año / siguiente) → Al año siguiente ya estaban divorciados.
(lunes / mañana) → El lunes por la mañana tuve un examen importante.
(año / invierno) → En el invierno de ese año empecé a nadar.


   * __________ (otoño / 2020) empecé a trabajar en mi primer empleo.
   * Fui a Italia __________ (verano / pasado) con mi familia.
   * __________ (mañana / ayer) estudié para el examen de matemáticas.
   * __________ (año / primavera) fue muy frío.
   * __________ (tarde / viernes) tenemos clase de español.
   * __________ (mañana / siguiente) fui a recoger mi comida.
