En portugués, es común que el pronombre reflexivo "se" siga a la preposición o el verbo, como en "deixaram de se falar". Sin embargo, en español, el pronombre "se" suele colocarse antes del verbo cuando se usa con preposiciones, como en "se dejaron de hablar", y detrás del verbo cuando este no está conjugado (p. ej., “volverse”, no “se volver”)..
Práctica
A continuación, tienes frases desordenadas. Pon sus elementos en el orden correcto según las reglas del español, colocando "se" en la posición adecuada.

   * decidir / el arte / se / dedicar / a
   * volver / se / una persona responsable / decidió
   * dejar / los amigos / con / se / de / ver
   * el problema / a / resolver / va / poco a poco / se
   * enamorar / poco a poco / ella / se / va / de / a / él
   * va / se / ahora / a / cada uno / su casa
   * lamenta / no / se / haber / con él / casado
