

* "Se trata de" es una construcción impersonal que se utiliza para definir o explicar algo o alguien, sin un sujeto específico. No se debe emplear cuando hay un sujeto claro en la oración.
* "Consiste en" se utiliza para explicar en qué consiste algo, es decir, para detallar los componentes o características de un sujeto o idea. Es adecuado usarlo cuando hay un sujeto claro que realiza la acción de consistir.
* “Trata” se usa cuando el sujeto realiza la acción de hablar, discutir o considerar un tema específico.
* “Trata de”, además de significar “intentar”, se emplea cuando se quiere hablar sobre el tema o la idea principal de algo.
Ejemplos
* Se trata de un método efectivo para aprender idiomas. (Impersonal, no hay un sujeto claro).
* La dieta mediterránea se trata de comer variado. (Hay un sujeto claro, "la dieta mediterránea", por lo que debería utilizarse "consiste en").
* La dieta mediterránea consiste en comer variado.
* El libro trata los efectos del calentamiento global.
* El seminario trata de las innovaciones tecnológicas.
* El seminario se trata de las innovaciones tecnológicas.
* En el programa radiofónico se han tratado las ventajas cognitivas de las personas bilingües.
* En el programa radiofónico se ha tratado de las ventajas cognitivas de las personas bilingües.
