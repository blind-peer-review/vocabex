En español, se usan expresiones como "mis primeros (X) años de vida", "mis años de infancia", o "mis años de juventud" para hablar de períodos o rangos de edad de manera más fluida y natural. También se usan términos como "niños pequeños" o "adolescentes" en lugar de mencionar el rango de edad numérico.
Práctica
Lee las siguientes oraciones y cámbialas para usar una expresión más natural sobre rangos de tiempo o edad. Luego, compáralas con las sugerencias correctas.

            * Cosas de edad 10-15 → Cosas de _________.
            * Libros para niños de edad 4-8 → Libros para _________.
            * Recuerdos de la edad 20-30 → Recuerdos de _________.
            * Juegos de edad 12-18 → Juegos de _________.
            * Ropa de edad 0-2 → Ropa de _________.
