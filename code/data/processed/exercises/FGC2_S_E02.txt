Rápido y pronto son dos palabras diferentes que se usan en situaciones distintas. Aquí tienes una explicación rápida para que puedas aprender cuándo usar cada una:
* Rápido: Se usa para hablar de la velocidad con la que ocurre algo. Ejemplo: "El coche va muy rápido" (el coche se mueve a gran velocidad).
* Pronto: Se usa para hablar de algo que va a ocurrir próximamente o en poco tiempo. Ejemplo: "Voy a terminar mis deberes pronto" (los terminaré dentro de poco tiempo).
Práctica
1. Corrige las frases siguientes cambiando rápido o pronto según sea necesario.
2. Escribe dos oraciones usando rápido y dos usando pronto para demostrar que has entendido la diferencia.


* Tengo que despertarme rápido mañana para no llegar tarde.
* El autobús llegó muy rápido, y pude irme a casa enseguida.
* Te llamo rápido, así hablamos de lo que pasó.
* Espero tu respuesta rápido, por favor.
* Camina pronto cuando está nerviosa.
