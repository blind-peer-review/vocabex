

La prefijación en español
* Palabras monoverbales: Se escribe todo junto sin espacio ni guion (p. ej. “auto” + “evaluación” = “autoevaluación”).
* Palabras polilexemáticas: El prefijo se escribe separado (p. ej. “ex” + “alto cargo” = “ex alto cargo”.
* Nombres propios y siglas: El prefijo se escribe separado mediante un guion (p. ej. “pos” + “Gorbachov” = “pos-Gorbachov”, “anti” + “OTAN” = anti-OTAN)
Práctica
Une cada prefijo con una palabra base, palabra compuesta o sigla para formar una nueva palabra o expresión. Luego, escribe una oración de ejemplo utilizando correctamente la palabra o expresión formada.
