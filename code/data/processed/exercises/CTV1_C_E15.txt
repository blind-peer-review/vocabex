A continuación, verás varias oraciones con expresiones relacionadas con la simultaneidad. Para cada una, elige la opción INCORRECTA.


                        * Los dos amigos terminaron la carrera (al mismo tiempo / a la vez / a la misma vez).
                        * No puedo hacer dos cosas (al mismo tiempo / a la vez / a la misma vez), necesito concentrarme en una sola.
                        * Los alumnos se levantaron (al mismo tiempo / a la vez / a la misma vez) que sonó la campana.
                        * Hablaban (al mismo tiempo / a la vez / a la misma vez)  y no podía entender nada.
