      * En  español, no existe la combinación "cq". Para los sonidos similares a "c" o "qu", usamos "c" o "qu" dependiendo de las vocales que siguen. La "qu" se usa antes de las vocales "e" o "i" para representar el sonido /k/. En cambio, la "c" se utiliza antes de las vocales "a", "o", "u" para el mismo sonido.
