Práctica
En cada grupo, identifica el sintagma que no corresponde, basándote en si la serie se refiere a disciplinas, a adjetivos relativos a esas disciplinas o a expertos en dichas disciplinas.


* Grupo 1: artículo psicológico / tesis antropológica / libro sociólogo / ensayo filosófico


* Grupo 2: psicólogo clínico / estudio lingüístico / historiador medieval / sociólogo urbano


* Grupo 3: enfermedad neurológica / psicología clínica / lingüística forense / economía financiera


* Grupo 4: charla antropológica / conferencia psicológica / presentación filosófica / exposición lingüista
