Práctica
Este ejercicio se centra en la práctica de traducciones literales coreano-español que son inadecuadas debido a las diferencias estructurales entre ambos idiomas. Lee las frases que se presentan a continuación. Tu tarea es corregir la traducción incorrecta al español para que resulte más idiomática.
Ejemplo
Coreano: “모든 가족이 식사를 위해 식탁 주위에서 모임을 합니다.”
Traducción incorrecta: “Todas las familias hacen reuniones alrededor de la mesa para comer.”
Corrección: “Todas las familias se reúnen alrededor de la mesa para comer.”
