Práctica

Para cada una, escribe una frase completa utilizando una de las siguientes expresiones para transmitir la urgencia o necesidad de la acción: "es importante que", "debe", "tiene que", "es necesario que", "es crucial que", "es esencial que". Puedes alterar la expresión según lo que consideres más adecuado para cada contexto.
