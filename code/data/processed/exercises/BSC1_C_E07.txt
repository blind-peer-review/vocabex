En español, cuando el objeto directo es una persona o un ser animado, usamos la preposición "a" para introducirlo.
Ejemplo
Correcto: Reuní a un grupo de chicas.
Incorrecto: Reuní un grupo de chicas.
Práctica
Elige la opción correcta para completar cada oración, teniendo en cuenta si es necesario usar la preposición "a" o no.
   1. Vi (a / Ø) los niños jugando en el parque.
   2. Ella buscó (a / Ø) su perro por todas partes.
   3. Visitamos (a / Ø) nuestros amigos en el hospital.
   4. Necesito encontrar (a / Ø) un buen restaurante.
   5. Escribo esta carta para notificar (a / Ø) las autoridades la explotación a la que está siendo sometida la plantilla de Mercadona.
   6. Llamaron (a / Ø) la policía para denunciar el robo.
