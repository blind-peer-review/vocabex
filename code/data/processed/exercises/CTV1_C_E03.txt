* En se utiliza para indicar que algo o alguien está dentro de un lugar temporal o espacial (por ejemplo: "en la ciudad", "en la escuela", “en primero de carrera”).
* De se utiliza para indicar procedencia o para especificar la relación de posesión o pertenencia a un lugar o categoría (por ejemplo: "la escuela de mi pueblo", "el hospital de Berlín").
