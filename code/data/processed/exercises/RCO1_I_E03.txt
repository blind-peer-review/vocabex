

Vamos a revisar el uso del conector “tanto… como…”, que se utiliza para expresar igualdad en la comparación entre dos elementos. Recuerda que:
* Cuando los elementos comparados son sintagmas preposicionales, el conector antecede la preposición (“la educación tanto de los niños como de los adolescentes”, no “la educación de tanto los niños como los adolescentes”).
* Cuando “tanto” antecede a un adverbio o un adjetivo, se usa la forma apocopada “tan”.
Ejemplos
* Miguel es tan alto como delgado.
* El manejo de los recursos naturales debe garantizar su disponibilidad tanto en la actualidad como en el futuro.
Práctica
A continuación, encontrarás frases desordenadas que incluyen la estructura “tanto… como”. Tu tarea es ordenar los elementos para formar oraciones coherentes.


1. bailar / le gusta / (tanto/tan) / música clásica / tocar el piano / como / a María.
2. necesario / para / una dieta equilibrada / (tanto/tan) / como / es / frutas / verduras / consumir.
3. son / como / los profesores / influyentes / los padres / (tanto/tan) / en la educación de los niños.
4. estudiar / como / importante / para / (tanto/tan) / es / dormir bien / tener buenos hábitos de estudio.
5. puede ser / como / (tanto/tan) / perjudicial / beneficioso / el uso de la tecnología.
6. para / grandes / de / pequeños / el espectáculo de fuegos artificiales / (tanto/tan) / como / fue / alegría / de / gratuito.
