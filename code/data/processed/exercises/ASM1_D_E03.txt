* En se utiliza para indicar que algo o alguien está dentro de un lugar o en una posición (por ejemplo: "en la ciudad", "en la escuela").
* De se utiliza para indicar procedencia o para especificar la relación de posesión o pertenencia a un lugar (por ejemplo: "la escuela de mi pueblo", "el hospital de Berlín").
Práctica
