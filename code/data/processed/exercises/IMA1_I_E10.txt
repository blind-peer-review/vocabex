Práctica
Para cada grupo, identifica el participio que está incorrectamente formado o que no encaja en el contexto proporcionado.


* Grupo 1: ha escrito una carta / descubrido un secreto / comido una manzana / visto una película / resuelto un misterio


* Grupo 2: ha rompido el récord / corrido una maratón / decidido ir de viaje / perdido las llaves / resuelto el problema.


* Grupo 3: ha prometido fidelidad / sido electo por su talento / estudio el tema / impreso el documento / cocinado una cena deliciosa


* Grupo 4: ha vendido la casa / traído flores para la ceremonia / cubierto las expectativas / suspendido la reunión / sido absolvido de los cargos.
